# Data Types

| C types | Name | Bits | Bytes |
| :----------- | :----------- | -----------: | -----------: |
| `char` | **Byte** | 8 | 1 |
| `short` | **Word** | 16 | 2 |
| `int`, `long` | **Doubleword** | 32 | 4 |
| `double`, `long long` | **Quadword** | 64 | 8 |
| n/a | **Double Quadword** | 128 | 16 |
